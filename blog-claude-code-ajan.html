Claude Code ile Sıfırdan AI Ajan Nasıl Kurulur — Gerçek Örneklerle

Kod bilmeden, sadece Claude Code kullanarak şirket süreçlerini otomatize eden yapay zeka ajanı nasıl kurarsın. Adım adım, gerçek örneklerle.

Burhan Kocabiyik
Coremagnet Kurucusu · Otomasyon & AI

"AI ajan" kelimesi çoğu kişide bir korku yaratıyor: karmaşık kod, Python bilgisi, API entegrasyonları... Aslında durum hiç de öyle değil.

Claude Code ile birkaç saat içinde gerçekten işe yarayan bir AI ajan kurabilirsin. Bu yazıda Coremagnet'te kullandığım örneklerle süreci adım adım anlatacağım.

AI Ajan Nedir, Ne İşe Yarar?

Bir AI ajan, belirli bir görevi otonom olarak yerine getiren ve bağımsız kararlar alabilen bir sistemdir. Normal otomasyon "A olursa B yap" der. AI ajan ise "müşteri mutlu değilse, sorunu tespit et ve çözüm öner" gibi bağlamsal kararlar alabilir.

Başlamadan Önce: İhtiyacın Olanlar

  • Claude Code erişimi (claude.ai/code üzerinden)
  • Anthropic API key (console.anthropic.com)
  • n8n kurulumu (n8n.io — ücretsiz cloud başlatabilirsin)
  • 15-20 dakika

💡 Kod yazmayacaksın. Claude Code senin için kodu üretiyor. Senin işin: ne istediğini anlatmak ve sonucu test etmek.

Adım 1: Kullanım Alanını Belirle

En sık yapılan hata: "Genel bir AI ajan yapayım" demek. Çok geniş bir kapsam başarısızlıkla sonuçlanır.

Doğru yaklaşım: Tek bir sorunu çöz. Örneğin: "Müşterilerden gelen destek e-postalarını kategorize et ve standart yanıtları otomatik gönder."

Bu kadar spesifik bir görev tanımıyla Claude Code'a gittiğinde, birkaç dakikada çalışan bir sistem kurabilirsin.

Adım 2: Claude Code'da Sistem Kurulumu

Claude Code'u aç ve şu şekilde bir prompt yaz:

Bir e-posta kategorileme ve otomatik yanıt sistemi kurmak istiyorum. Sistem şunları yapmalı: - Gmail'den gelen e-postaları her saat kontrol et - E-postayı analiz et ve kategorize et: (fiyat sorusu / teknik destek / iade talebi / diğer) - Fiyat sorusu ve teknik destek için hazır şablondan yanıt ver - İade talepleri ve "diğer" kategorisini bana flag et Kullandığım araçlar: n8n, Gmail API, Anthropic Claude API Başlangıç için n8n workflow JSON'unu oluşturur musun?

Claude Code bu prompt'a karşılık sana tam n8n workflow JSON'unu verir. Kopyala, n8n'e import et, API key'lerini gir — çalışır.

Adım 3: n8n'e Entegre Et

Claude Code'un ürettiği workflow'u n8n'e aktarmak için:

  1. n8n dashboard'unu aç → "New Workflow" → "Import from JSON"
  2. Claude Code'un ürettiği JSON'u yapıştır
  3. Gmail node'una Gmail OAuth credentials ekle
  4. HTTP Request node'una Claude API key'ini ekle
  5. Test et: "Execute Workflow" butonuna bas

🔑 API key'i n8n'de credentials olarak sakla — workflow içine düz metin olarak yapıştırma. n8n'in Credentials menüsü bu iş için var.

Adım 4: Prompt'u Şirketine Özelleştir

n8n içindeki Claude API node'una gönderdiğin "system prompt" — ajanının kişiliğini ve davranışını belirler. Bunu şirketine göre özelleştir.

Sen Coremagnet'in müşteri destek asistanısın. Görevin gelen e-postaları analiz etmek ve şu JSON formatında yanıt vermek: { "kategori": "fiyat_sorusu | teknik_destek | iade | diger", "oncelik": "yuksek | normal | dusuk", "otomatik_yanit": true/false, "yanit_taslagi": "eğer otomatik yanıt verilecekse buraya yaz" } Şirket hakkında bilgi: - Otomasyon ajansı, n8n ve AI çözümleri sunuyoruz - Fiyatlandırma: [fiyat bilgisi] - Teknik destek: destek@coremagnet.com

Gerçek Örnek: Coremagnet'te Ne Yaptım

Coremagnet'te bu sistemi kurduğumda, haftalık gelen ~80 e-postanın %65'i otomatik yanıtlanır hale geldi. Geri kalan %35 ise sadece gerçekten dikkat gerektiren sorulardı.

Sonuç: Müşteri desteğine harcadığım haftalık 6 saat, 2 saate düştü. Ve yanıt kalitesi arttı — çünkü insan dikkati gerçekten zor sorulara gidiyor.

Dikkat Etmen Gereken 3 Şey

1. Her zaman insan denetimi bırak

AI ajan mükemmel değil. Kritik kararlar için her zaman bir "insan onayı" adımı ekle. n8n'in "Wait" node'u bu iş için mükemmel.

2. Prompt'u test et, optimize et

İlk prompt mükemmel olmaz. 10-20 gerçek e-postayla test et, hatalı kategorilere bak, prompt'u güncelle. Bu iterasyon süreci kritik.

3. Maliyetleri takip et

Claude API kullanımı ücretli. n8n üzerindeki her workflow çalışması bir API çağrısı. Başta düşük volume'la test et, maliyeti anla, sonra ölçekle.

💸 Tipik maliyet: 1.000 e-posta analizi için ~$2-5 arası. Kazandırılan zamana kıyasla bu çok küçük bir rakam.

Özet: Başlamak İçin Yapman Gereken 3 Şey

  1. Tek bir süreci seç — en çok zaman alan tekrarlayan işi belirle
  2. Claude Code'a anlat — ne yapmasını istediğini net şekilde yaz, n8n workflow iste
  3. Test et ve iterate et — mükemmel olmayacak, ama ilk iterasyon bile %60-70 zaman kazandırır

AI ajan kurulumu artık yazılımcıların tekeli değil. Doğru araçlarla, doğru prompt'larla, herkes bunu yapabilir.
